# JD Higienização de Estofados

Site profissional para serviços de higienização de estofados a domicílio.

## 🌟 Sobre o Projeto

Site institucional da JD Higienização de Estofados, oferecendo serviços profissionais de limpeza e higienização de:
- Sofás e estofados
- Colchões
- Carpetes e tapetes
- Higienização automotiva
- Cadeiras e poltronas
- Serviços empresariais

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Poppins)

## 📁 Estrutura do Projeto

## 🔧 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/Alfredoprogramador/JD-HIGIENIZA-O-DE-ESTOFADOS.git
```

2. Abra o arquivo `index.html` no navegador

Ou hospede em qualquer servidor web (GitHub Pages, Netlify, Vercel, etc.)

## 📱 Personalização

### Informações de Contato
Edite o arquivo `index.html` nas seções:
- Telefone/WhatsApp (linha ~126)
- Email (linha ~132)
- Localização (linha ~138)

### Cores do Site
Edite o arquivo `css/style.css` nas variáveis CSS (linhas 9-18):
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    /* ... outras cores */
}
```

### WhatsApp
Configure o número do WhatsApp no arquivo `js/script.js` (linha 71):
```javascript
const numeroWhatsApp = '5500000000000'; // Seu número com DDI + DDD
```

## 🌐 Deploy para Produção

### GitHub Pages (Grátis)
1. Vá em Settings → Pages
2. Source: Deploy from branch
3. Branch: main / (root)
4. Salve

Seu site estará em: `https://alfredoprogramador.github.io/JD-HIGIENIZA-O-DE-ESTOFADOS/`

### Netlify (Grátis)
1. Acesse [netlify.com](https://netlify.com)
2. Conecte seu repositório GitHub
3. Deploy automático

### Vercel (Grátis)
1. Acesse [vercel.com](https://vercel.com)
2. Import Git Repository
3. Deploy
